### 🛠️ What I'm building

- **[vektordb](https://github.com/MaheshBhushan/vektordb)** — a vector database written
  from first principles in Rust: HNSW indexing, mmap storage, product quantization, AVX2
  SIMD kernels, lock-free reads, and a write-ahead log. Basically "what if I understood
  every layer of the thing I keep importing."
- **[career-ops](https://github.com/MaheshBhushan/career-ops)** — an AI-powered job-search
  system built on Claude Code: 14 skill modes, a Go dashboard, PDF generation, and batch
  processing.
- **[article-image-generator](https://github.com/MaheshBhushan/article-image-generator)**
  — turns an article description into product imagery with Stable Diffusion XL.
- **[mk-fuzz](https://github.com/MaheshBhushan/mk-fuzz)** — a coverage-guided fuzzer in C,
  because sometimes you want to break your own code before anyone else does.
- **[FinanRAG](https://github.com/MaheshBhushan/FinanRAG)** — retrieval-augmented
  generation over financial documents.
